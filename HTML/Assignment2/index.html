                In inline element all the elements appred in same line with a single space in between them.
                In block element all the appred one below the other taking whole width of parent element.
                In inline-block element they take the width as set.
                The block elements take up the full width available.
                The inline elements take the required width to display the contents of the elements.
                Inline-block allows to set a width and height on the element.
